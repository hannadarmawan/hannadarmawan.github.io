Deskripsi Singkat
Website Jurusan Ilmu Pemerintahan FISIP UPR

Tujuan dari Rancang Bangun Website Jurusan Ilmu Pemerintahan Fakultas Ilmu Sosial dan Ilmu Politik Universitas Palangka Raya ini adalah menjadi salah satu media informasi yang berguna sebagai media penyampaian informasi bagi dosen, mahasiswa, pengunjung serta dapat melakukan pengelolaan data akademik pada Jurusan Ilmu Pemerintahan Fakultas Ilmu Sosial dan Ilmu Politik (FISIP) Universitas Palangka Raya.

Fitur :

  • Profil Jurusan
      • Sejarah, Visi Misi, Kompetensi, Tujuan, Struktur & Fasilitas
  • Akademik
      • Profil Dosen, Data Mahasiswa, Mata Kuliah, Jadwal Perkuliahan, Nilai Mahasiswa KRS Online, KHS Online, Transkrip Nilai, Absensi.
  • Kalender Akademik, Jurnal, Praktikum TIK, Bahan Ajar, Gallery, Buku Tamu

    • Software (Perangkat Lunak) :
    Xampp sebagai web server, Untuk validasi menggunakan PHP Script dan Java Script, MySQL sebagai Database, Adobe Dreamweaver dan Notepad++ sebagai pengolahan script, Adobe Photoshop sebagai editing gambar, Google Chrome sebagai web browser, Untuk pemodelan desain Unified Modeling Language (UML), visualisai dengan menggunakan Edraw Max 7.2.

    UML-based Web Engineering (UWE)

    Metode yang digunakan dalam pembuatan “Website Jurusan Ilmu Pemerintahan Fakultas Ilmu Sosial dan Ilmu Politik Universitas Palangka Raya” dengan metodologi UML-based Web Engineering (UWE) dimana merupakan pendekatan metodologi untuk pengembangan aplikasi web yang berdasarkan pada Unified Process (Jacobsen, dkk, 2000).

  • Requirements Model (Use Case Model)
      • Use Case Diagram, Activity Diagram
  • Content Model (Conceptual Model)
      • Class Diagram, State Chart Diagram
  • Navigation Model
  • Presentation Model

    • Berikut adalah UML Diagram untuk pengguna sebagain Admin dari Website Website Jurusan Ilmu Pemerintahan FISIP UPR. Selain pengunjung ada Dosen Koord Mata Kuliah, Tim Redaksi Jurnal, Mahasiswa & Pengunjung
